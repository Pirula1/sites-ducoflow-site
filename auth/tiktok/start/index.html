TikTok Login Kit

This page starts the TikTok Login Kit flow for DucoFlow.

Authorized users can use TikTok Login to securely connect their account and grant access to approved DucoFlow workspace features.

This flow is used to identify the authorized TikTok account, retrieve public profile information, display account metrics, and list authorized public videos.
